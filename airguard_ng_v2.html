Setup Instructions

1
Wire your ESP32: MQ2 analog out → GPIO34, DHT11 data → GPIO4, both sensors to 3.3V and GND.
2
Run pip install flask then start the receiver: python esp32_receiver.py on your laptop.
3
Find your laptop IP: run ipconfig in terminal, look for IPv4 Address (e.g. 192.168.8.101).
4
In the Arduino code (airguard_device.ino), paste your WiFi name, password, and laptop IP into the config section at the top.
5
Upload the code to your ESP32. Both devices must be on the same WiFi network. The dashboard auto-refreshes every 30 seconds.
